# Rollout scorecard review

Use this runbook when reviewing the scorecards that govern rollout expansion after
the bounded pilot.

## Workflow

1. open `meta/rollout-scorecard-program.yaml`
2. confirm every scorecard has explicit metrics, evidence refs, and review docs
3. compare the scorecards against the current pilot and rollout evidence bundles
4. fail the review if a scorecard relies on anecdotal success claims alone
