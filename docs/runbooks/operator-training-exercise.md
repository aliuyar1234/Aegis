# Operator training exercise

Use this runbook when running a bounded PHASE-19 operator exercise.

## Required inputs

- selected exercise id
- referenced runbooks
- required evidence bundles

## Exercise loop

1. confirm the exercise prerequisites and required evidence exist
2. review the linked runbooks with the operator cohort
3. execute the bounded exercise steps
4. verify the expected outcomes are reached
5. record the completed adoption evidence bundle

## Boundaries

- exercises must stay inside committed runbooks
- exercises do not authorize runtime bypasses
- failed exercises require remediation through the referenced runbooks, not local shortcuts
