# Customer operations handshake

Use this runbook when handing the customer-facing operations package to a design
partner or pilot operator.

## Workflow

1. open `meta/customer-operations-package.yaml`
2. verify installation, deployment, upgrades, limits, failure modes, and responsibilities are all covered
3. confirm the customer-facing docs are current and consistent with support policy
4. attach the package to the pilot handoff before launch
