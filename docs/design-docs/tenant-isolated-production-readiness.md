# Tenant-isolated production readiness

This document defines the minimum production-readiness and migration-boundary surface
required when Aegis moves beyond pilot-shaped customer operations.

## Required surfaces

- tenant-isolated deployment boundaries
- migration entry criteria
- rollback and restore boundaries
- dedicated-tenant evidence inputs

## Constraint

Post-pilot customer launches may not treat tenant-isolated production as a pilot
exception. Production isolation must be explicit enough to review from committed
evidence and runbooks alone.
