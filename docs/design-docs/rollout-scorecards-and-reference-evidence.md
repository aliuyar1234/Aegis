# Rollout scorecards and reference evidence

This document defines the scorecard surface that governs rollout expansion after
the first bounded pilot wave.

## Required surfaces

- post-pilot rollout scorecards
- reference-customer evidence inputs
- expansion metrics
- scorecard review runbooks

## Constraint

Rollout expansion must not be justified by anecdotal success claims alone. Each
scorecard has to bind explicit evidence, metrics, and review surfaces.
