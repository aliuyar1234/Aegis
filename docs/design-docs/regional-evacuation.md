# Regional evacuation

## Purpose

Regional evacuation is the bounded PHASE-17 control surface for moving authority away
from an impaired or unsuitable region.

## Required drill shape

- fence the source region
- verify restore evidence
- switch regional placement policy
- promote the target region
- verify single-owner and continuity expectations

## Canonical machine-readable artifact

- `meta/regional-evacuation-fixtures.yaml`

## Anti-patterns

- evacuation without explicit restore evidence
- assuming session continuity without checkpoint and artifact lineage checks
- treating operator-run evacuation as an undocumented emergency-only path
