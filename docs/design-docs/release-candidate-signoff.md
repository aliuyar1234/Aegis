# Release candidate signoff

This document defines the release signoff surface for customer launch readiness.

## Why this exists

PHASE-15 established release evidence. Customer launch readiness needs a narrower
operator-facing signoff contract that binds that evidence to restore drills, rollback
runbooks, and the supported evaluation workflow.

## Required signoff inputs

- current release evidence bundle
- restore drill references
- rollback runbooks
- evaluation workflow target names
- support-facing launch docs

## Constraint

Signoff is not complete if any required evidence or rollback surface is implicit.
