# Service-scale operations readiness

This document defines the operating cadence required when Aegis supports more than
one bounded pilot customer at a time.

## Required surfaces

- service-scale support cadences
- owning support rotations
- escalation coverage beyond pilot launch week
- required observability and communication surfaces

## Constraint

Service-scale operations must not extend founder-led pilot support indefinitely.
The multi-customer launch posture has to be explicit enough for new operators to
run from committed docs and rotations.
