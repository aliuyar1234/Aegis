# Launch security baseline

This document defines the minimum security control surface required before Aegis
is treated as customer-launch ready.

## Required control categories

- dependency audit
- SBOM generation
- secret scanning
- signed build artifacts
- hardened defaults
- scoped credentials
- key rotation

## Constraint

Security baseline controls must be represented as explicit launch controls with
owners, doc references, and review runbooks. They may not live only as broad
security aspirations.
